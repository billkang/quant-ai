# 事件查询

事件查询模块用于监控和分析可能影响市场的宏观、行业及个股事件。

> **入口变更**：事件查询已整合至「数据管理」模块下的「事件查询」标签页，也可通过左侧导航栏的「数据管理」进入。

## 事件列表

展示系统采集到的各类事件：
- **宏观事件**：央行政策、GDP/CPI 数据发布、利率决议等
- **行业事件**：行业政策、技术突破、价格变动等
- **公司事件**：财报发布、分红送转、重大事项公告等

每条事件包含：
- 事件类型与标题
- 发生时间
- 关联股票/行业
- 事件等级（高/中/低）

## 筛选与搜索

- 按事件类型筛选
- 按时间范围筛选
- 按关联股票搜索
- 按等级筛选

## 事件详情

点击事件标题查看详情：
- 事件摘要
- 关联股票列表
- 历史同类事件回顾
- 市场预期与实际影响对比

## 数据源与采集

事件数据来源于「数据管理」-「渠道管理」中配置的外部采集渠道，包括：
- 财经新闻网站
- 交易所公告接口
- 宏观经济数据 API

在「数据管理」-「采集监控」页面可查看事件数据的定时拉取状态：
- 任务名称与执行周期
- 上次执行时间与结果
- 失败重试机制

## 使用技巧

- 关注高等级宏观事件，提前调整仓位
- 公司财报季使用事件查询跟踪持仓标的
- 结合「规则管理」设置事件触发的自动告警
