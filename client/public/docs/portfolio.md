# 资产组合

资产组合模块帮助用户分析持仓结构、评估组合风险与收益。

## 组合概览

展示当前持仓的核心指标：
- **总资产**：各持仓市值加总
- **累计收益**：持仓盈亏合计
- **今日收益**：当日浮动盈亏
- **收益率**：累计收益 / 成本

## 持仓列表

逐条展示每只持仓股票：
- 股票代码与名称
- 持仓数量与成本价
- 最新市值与盈亏
- 持仓占比

**操作**：
- 点击股票行进入详情页
- 支持手动添加/编辑/删除持仓记录

## 行业分布

以饼图形式展示持仓在各行业的占比，帮助识别集中度风险。

## 收益归因

分析组合收益的来源：
- 个股贡献度
- 行业贡献度
- 时间维度收益分解

## 组合分析 API

系统每日自动运行组合分析任务，计算：
- 各股票的技术指标状态
- 估值水平（PE、PB 分位）
- 风险评分

分析结果在 **仪表盘** 的快捷入口中查看。

## 使用技巧

- 持仓数据可通过虚拟盘交易自动同步，也可手动维护
- 行业分布饼图支持点击下钻查看具体股票
- 建议定期查看收益归因，及时调整权重过高的标的
