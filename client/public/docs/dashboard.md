# 仪表盘

仪表盘是 QuantMaster 的默认首页，提供全局概览与快捷操作。

## 功能区域

### 1. 统计卡片
页面顶部展示四项核心指标：
- **策略数量** — 当前创建/收藏的策略总数
- **回测次数** — 累计执行的历史回测次数
- **自选数量** — 自选股列表中的股票数量
- **数据覆盖** — 价格/指标/事件三类数据的覆盖情况

### 2. 自选股列表
实时展示已添加自选的股票：
- 股票代码与名称
- 最新价格
- 涨跌幅（红色上涨，绿色下跌）
- 涨跌幅百分比

**操作**：
- 点击股票行进入 **行情分析** 详情页
- 点击 ] 删除该股票
- 点击顶部搜索框输入代码或名称添加新股票

### 3. 最近回测
展示最近执行的 5 条回测记录，包含策略名称、时间区间与收益率。

### 4. 快捷入口
提供常用功能的快速跳转按钮：策略管理、回测报告、数据管理等。

## 使用技巧

- 自选股数据通过 WebSocket 实时推送，无需手动刷新
- 点击统计卡片可直接跳转到对应模块
- 使用搜索框支持模糊匹配股票代码和名称
