# 行情分析

行情分析模块提供个股深度数据与可视化图表，辅助交易决策。

## 股票详情页

点击仪表盘自选股或搜索进入股票详情，包含：

### 基本信息
- 股票代码、名称、所属行业
- 最新价、涨跌幅、成交量
- 市值、市盈率、市净率等基本面指标

### K线图表
- 支持日K、周K、月K切换
- 叠加均线（MA5/MA10/MA20/MA60）
- 成交量柱状图

### 技术指标
实时计算的常用技术指标：
| 指标 | 说明 |
|------|------|
| MA | 移动平均线，判断趋势方向 |
| RSI | 相对强弱指数，0-100 衡量超买超卖 |
| MACD | 异同移动平均线，判断趋势转折 |
| KDJ | 随机指标，捕捉短线买卖点 |
| BOLL | 布林带，判断价格波动区间 |

## 使用技巧

- 在图表区域鼠标悬停可查看具体日期的 OHLC 数据
- 指标数据每日收盘后自动更新
- 支持将当前股票一键加入自选股
