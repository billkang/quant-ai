# 策略管理

策略管理模块用于创建、编辑和维护自定义量化策略。

## 创建策略

1. 进入 **策略管理** 页面
2. 点击 **新建策略**
3. 填写策略信息：
   - **策略名称**：便于识别的名称
   - **策略代码**：Python 函数实现，需符合模板规范
   - **参数模式**：定义可外部调整的参数（如均线周期）

### 策略代码规范

策略函数接收以下参数：
```python
def strategy(ctx, code, start_date, end_date, params):
    # ctx: 数据上下文，提供行情接口
    # code: 股票代码
    # start_date / end_date: 回测区间
    # params: 用户传入的参数字典
    
    df = ctx.get_price(code, start_date, end_date)
    # 实现买卖逻辑，返回信号序列
    return signals
```

## 编辑与删除

- 点击策略卡片上的 **编辑** 修改代码或参数
- 点击 **删除** 移除不再使用的策略（内置策略不可删除）

## 策略库

**策略库** 页面展示系统内置的经典策略：
- 双均线交叉（金叉买入、死叉卖出）
- MACD 趋势跟踪
- RSI 超买超卖反转
- 布林带突破

可以直接 **收藏** 内置策略到个人策略列表，或在回测中直接引用。

## 使用技巧

- 策略代码修改后，历史回测记录不会自动更新
- 建议先在 **回测报告** 中验证策略有效性，再进入虚拟盘
- 参数模式使用 JSON Schema 格式，支持数字、字符串、枚举类型
