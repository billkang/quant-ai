# QuantMaster 使用手册

欢迎使用 **QuantMaster** —— 一款面向量化交易者的智能投研平台。

## 平台简介

QuantMaster 集成了行情数据、技术指标、策略回测、资产组合分析、虚拟盘交易、事件驱动监控等核心功能，帮助用户从数据中发现交易机会，并用科学的方法验证策略有效性。

## 核心功能一览

| 模块 | 说明 |
|------|------|
| **仪表盘** | 自选股监控、统计概览、快捷入口 |
| **行情分析** | 个股详情、K线图表、实时技术指标 |
| **策略管理** | 自定义策略的创建、编辑与参数配置 |
| **策略库** | 内置经典策略（均线、MACD、RSI 等） |
| **回测报告** | 策略历史回测、收益曲线、风险指标 |
| **资产组合** | 持仓分析、行业分布、收益归因 |
| **虚拟盘** | 模拟真实交易环境，零风险验证策略 |
| **事件查询** | 宏观经济、行业事件与个股关联监控 |
| **数据源配置** | 管理数据采集来源与接口 |
| **采集任务** | 定时数据拉取任务的管理与调度 |
| **规则管理** | 价格、指标、事件告警规则配置 |
| **数据管理** | 数据质量检查、补录与维护 |
| **系统设置** | 主题切换、通知偏好、账户管理 |

## 快速开始

1. 登录系统后进入 **仪表盘**
2. 点击左上角搜索框添加自选股
3. 进入 **策略库** 选择一个内置策略
4. 在 **回测报告** 中验证策略历史表现
5. 使用 **虚拟盘** 进行模拟交易

## 需要帮助？

- 点击任意页面右上角的 ] 图标查看对应模块的说明
- 使用左侧导航快速切换功能模块
- 所有数据均为实时或准实时更新
