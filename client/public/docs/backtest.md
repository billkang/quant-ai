# 回测报告

回测报告模块用于在历史数据上验证策略表现，是策略上线前的核心验证环节。

## 执行回测

1. 进入 **回测报告** 页面
2. 选择策略（自定义策略或内置策略）
3. 配置参数：
   - **股票代码**：回测标的
   - **开始日期 / 结束日期**：回测时间区间（建议至少 1 年）
   - **初始资金**：模拟起始资金（默认 10 万）
   - **策略参数**：根据策略模式动态生成的表单
4. 点击 **开始回测**

## 回测结果

### 收益曲线
展示策略累计收益率与基准（如沪深 300）的对比曲线。

### 交易记录
列出回测期间的所有买卖操作：
- 成交日期
- 操作类型（买入/卖出）
- 成交价格与数量
- 手续费

### 风险指标
| 指标 | 说明 |
|------|------|
| 总收益率 | 整个区间的累计收益 |
| 年化收益率 | 按复利折算的年化收益 |
| 最大回撤 | 从峰值到谷底的最大亏损幅度 |
| 夏普比率 | 单位风险带来的超额收益 |
| 胜率 | 盈利交易次数 / 总交易次数 |
| 盈亏比 | 平均盈利 / 平均亏损 |

## 历史记录

所有回测结果自动保存，支持：
- 按策略名称筛选
- 查看过往回测的详细报告
- 删除不再需要的记录

## 使用技巧

- 回测区间应覆盖牛熊周期，避免过拟合
- 同一策略不同参数可保存多条记录进行对比
- 结果仅供参考，历史表现不代表未来收益
