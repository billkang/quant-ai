# 股票筛选器

股票筛选器支持通过多维度条件快速找出符合要求的标的。

## 筛选条件

支持以下字段的条件组合：
| 字段 | 说明 | 常用运算符 |
|------|------|-----------|
| PE(TTM) | 滚动市盈率 | 小于、大于、区间 |
| PB | 市净率 | 小于、大于、区间 |
| ROE(%) | 净资产收益率 | 大于 |
| RSI6 | 6日相对强弱指标 | 小于（超卖）、大于（超买） |
| RSI12 | 12日 RSI | 同上 |
| RSI24 | 24日 RSI | 同上 |
| 负债率(%) | 资产负债率 | 小于 |
| 营收增速(%) | 营业收入同比增长 | 大于 |
| 净利润增速(%) | 净利润同比增长 | 大于 |

## 添加条件

1. 选择字段
2. 选择运算符（小于、大于、等于、区间）
3. 输入数值
4. 点击 **添加条件**

支持同时添加多个条件，条件之间为 **且** 关系。

## 执行筛选

点击 **开始筛选** 后，系统返回符合条件的股票列表：
- 股票代码与名称
- 最新价格
- 各条件字段的实际值

## 保存筛选方案

- 点击 **保存方案**，输入名称
- 已保存的方案可在下拉框中快速切换
- 支持删除不再需要的方案

## 使用技巧

- 建议先宽后窄，逐步增加条件观察结果变化
- 可将常用方案保存，如 "低估值高成长"、"RSI超卖反弹" 等
- 筛选结果支持导出，方便进一步分析
