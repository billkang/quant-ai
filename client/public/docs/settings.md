# 系统设置

系统设置模块用于管理个人偏好、通知配置和账户信息。

## 主题切换

QuantMaster 支持多种主题：
| 主题 | 风格 |
|------|------|
| 海洋蓝 | 清爽明亮的蓝色调 |
| 晨曦白 | 简洁专业的浅色主题 |
| 午夜黑 | 深色模式，适合夜间使用 |

切换方式：
- 点击顶部导航栏右侧的主题色块
- 选择喜欢的主题即时生效

## 通知设置

配置各类通知的接收方式：
- **价格告警**：站内/邮件/Webhook
- **指标告警**：站内/邮件/Webhook
- **事件告警**：站内/邮件/Webhook
- **系统通知**：维护公告、功能更新

## Webhook 配置

如需将告警推送到企业微信/钉钉/飞书：
1. 在对应平台创建群机器人，获取 Webhook 地址
2. 在系统设置中粘贴地址
3. 选择推送格式（Markdown/纯文本）
4. 点击 **测试推送** 验证连通性

## 账户管理

- 查看当前登录用户名
- 修改密码
- 登出账户

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| / | 聚焦搜索框 |
| Esc | 关闭弹窗/取消操作 |

## 使用技巧

- 深色主题在夜间使用时更护眼
- 建议开启重要告警的邮件通知，避免遗漏
- Webhook 推送支持 @全体成员，适合交易团队使用
