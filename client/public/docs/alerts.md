# 告警与规则管理

告警系统帮助用户实时监控市场变化，当满足预设条件时自动推送通知。

## 规则管理

在 **规则管理** 页面配置告警规则：

### 规则类型
| 类型 | 说明 |
|------|------|
| 价格告警 | 股价突破或跌破设定值 |
| 涨跌幅告警 | 单日涨跌幅度超过阈值 |
| 指标告警 | RSI、MACD 等技术指标达到条件 |
| 事件告警 | 特定类型事件发生时触发 |

### 创建规则
1. 选择规则类型
2. 设置监控标的（单股或组合）
3. 配置触发条件与阈值
4. 选择通知方式（站内消息/邮件）
5. 设置生效时间范围

## 告警中心

点击顶部导航栏的 **] 图标进入告警中心：
- 未读告警以红点数字提醒
- 支持按类型、时间、已读状态筛选
- 点击告警项可跳转至相关页面

## 通知方式

- **站内消息**：应用内实时推送
- **邮件通知**：发送告警详情到注册邮箱
- **Webhook**：支持推送到企业微信/钉钉/飞书（需在系统设置中配置）

## 使用技巧

- 避免设置过于敏感的阈值，防止告警轰炸
- 对重要持仓建议设置价格突破与指标背离双重告警
- 事件告警适合财报季、政策发布窗口期使用
