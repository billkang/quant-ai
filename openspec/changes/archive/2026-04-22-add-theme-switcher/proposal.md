## Why

当前系统仅有一套蓝色深色主题，视觉风格单一且长时间使用易产生疲劳。用户希望能根据个人偏好和使用场景切换不同主题风格（如清亮白天模式、专业深色模式），提升产品的可用性和视觉体验。

## What Changes

- 引入三套精心设计的主题配色方案，每套主题配有中文名称：
  - **「深海蓝」** — 保留现有蓝色主题作为默认主题
  - **「晨曦白」** — 清亮浅色主题，适合白天使用
  - **「极夜黑」** — 纯黑深色主题，专业沉稳
- 在页面布局中添加主题切换控件（设置按钮 + 下拉选择器）
- 将现有的硬编码 CSS 变量替换为 CSS 自定义属性（var），通过切换 `data-theme` 属性实现主题切换
- 将主题选择持久化到 `localStorage`，刷新页面后保持用户偏好
- 同步适配 Ant Design 的主题 token，确保组件库与自定义样式风格一致

## Capabilities

### New Capabilities
- `theme-switcher`: 多主题切换与持久化能力，包括主题定义、切换 UI、持久化存储

### Modified Capabilities
- *(无现有 spec 需要修改 — 本变更仅涉及前端视觉层，不改动 API 或数据模型)*

## Impact

- **前端**: `client/src/index.css`（变量定义重构）、`client/src/components/Layout.tsx`（添加切换控件）、新增主题配置文件
- **UI 组件库**: Ant Design 主题 token 需要随主题动态注入
- **无后端影响**: 不涉及 API、数据库或业务逻辑变更
